**博客目的**--*情不知所起，一往而深*
-------
此博客的内容主要是记录某位西交的本科生*所见所闻所感所想*，在高中时对于大学的生活很是向往，认为一到了大学便会脱离高中那些无聊的题目、令人厌倦的学习生活。梦想着大学的生活是一种解放，是一种自由。然而幻想终究只是幻想，在上了大学的第一个月后，此种幻想便不复存在了。

>到不了的都叫做远方，回不去的名字叫故乡――*《牡丹江》*

所谓大学，并非一个供人享乐的地方，在逐渐适应大学的生活中我发现：我们上大学的目的有如下几点：

- 为了学习专业技能
- 为了认识你自己

其中第一点原因显而易见，我们通过一系列的课程提高自身专业技能，而其中的第二点
引用一下一句名言，但是不知道出处了
>一万年太久，让我们只争朝夕，不负韶华
